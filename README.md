# Olist Brazilian E-Commerce Analysis

## Project Overview
End-to-end SQL data analysis on Brazil's largest e-commerce dataset (Olist).
Analyzed 100,000+ orders across 9 relational tables to uncover revenue trends,
customer behavior, and delivery performance insights.

## Tools Used
- MySQL — data storage and SQL analysis
- Tableau Public — dashboard and visualization
- GitHub — version control

## Dataset
- Source: Kaggle - Olist Brazilian E-Commerce
- 9 tables, 600,000+ rows
- Period: September 2016 to October 2018

## Live Dashboard
https://public.tableau.com/app/profile/rahul.mandal3458/viz/OlistBrazilianE-CommerceAnalysis_17822842204000/Dashboard1

## Key Findings
1. Fast delivery scores 4.41 vs 3.06 for very slow delivery
2. Only 3% of customers return - major retention opportunity
3. Champion customers spend 2x more than average
4. 10-installment buyers spend 4x more than full-payment buyers
5. Sao Paulo generates 37% of total revenue
6. Health and Beauty is the top revenue category at R$1.4M

## SQL Techniques Used
- Multi-table JOINs across 9 tables
- CTEs (Common Table Expressions)
- Window Functions (LAG, RANK, NTILE, SUM OVER)
- RFM Segmentation
- CASE statements and subqueries
- Date functions and aggregations
